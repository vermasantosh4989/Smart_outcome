# Student Performance Prediction using Ensemble Machine Learning

## Project Overview

This project predicts **course completion status (classification)** and
**final score (regression)** using multiple ensemble learning
algorithms. It includes preprocessing, EDA, Bagging, AdaBoost, Gradient
Boosting, LightGBM, XGBoost, Voting, and Stacking, followed by
performance comparison.

## Objectives

-   Preprocess and clean the dataset.
-   Build classification and regression models.
-   Compare ensemble techniques.
-   Evaluate models using standard metrics.

## Workflow

1.  Data Preprocessing
2.  Exploratory Data Analysis (EDA)
3.  Bagging
4.  AdaBoost
5.  Gradient Boosting
6.  LightGBM
7.  XGBoost
8.  Voting & Stacking
9.  Model Comparison

## Classification Metrics

-   Accuracy
-   Precision
-   Recall

-   ROC-AUC
-   Confusion Matrix
-   ROC Curve

## Regression Metrics

-   MAE
-   RMSE
-   R² Score

## Conclusion

This project compares multiple ensemble learning algorithms for
classification and regression and identifies the best-performing model
using standard evaluation metrics.
